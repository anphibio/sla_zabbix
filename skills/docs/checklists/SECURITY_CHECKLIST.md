# Checklist de Segurança

- [ ] Não há secrets hardcoded
- [ ] `.env` real não está versionado
- [ ] Inputs são validados
- [ ] Permissões são validadas no backend
- [ ] Logs não expõem dados sensíveis
- [ ] Endpoints privados exigem autenticação
- [ ] Endpoints administrativos exigem autorização
- [ ] Erros não expõem stack trace
- [ ] CORS está restrito
- [ ] Rate limit avaliado
- [ ] Superpowers usado quando aplicável
