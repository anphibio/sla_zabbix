# CREATE_FEATURE

Objetivo:

Implementar uma nova funcionalidade respeitando todos os arquivos de governança.

Antes de implementar:

1. Explique o objetivo da funcionalidade.
2. Identifique arquivos impactados.
3. Identifique impacto no banco.
4. Identifique impacto na API.
5. Identifique impacto na UI.
6. Identifique riscos de segurança.
7. Identifique impacto em observabilidade.
8. Identifique se Superpowers deve ser usado.
9. Crie plano curto.

Durante a implementação:

- Atualize backend
- Atualize frontend quando aplicável
- Crie migration quando necessário
- Crie testes
- Atualize documentação
- Adicione logs, auditoria e métricas quando aplicável

Ao final:

- Liste arquivos alterados
- Informe comandos de teste
- Informe riscos restantes
