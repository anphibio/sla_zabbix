# FIX_BUG

Objetivo:

Corrigir bug com análise de causa raiz.

Use Superpowers preferencialmente se disponível.

Passos:

1. Reproduza ou explique como reproduzir.
2. Identifique causa raiz.
3. Corrija sem quebrar funcionalidades existentes.
4. Crie teste de regressão.
5. Valide impacto.
6. Verifique se logs ou métricas ajudariam a detectar o bug no futuro.
7. Informe arquivos alterados.
