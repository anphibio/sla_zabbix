# IMPECCABLE_REVIEW

Revise a interface criada utilizando os princípios Impeccable.

Use Superpowers preferencialmente se houver skill aplicável.

Avalie:

- Hierarquia visual
- Espaçamento
- Tipografia
- Contraste
- Consistência
- Responsividade
- UX
- Estados vazios
- Estados de erro
- Loading
- Ações destrutivas
- Feedbacks
- Acessibilidade

Atribua nota de 0 a 10.

Não implemente código nesta etapa.
Apenas critique e sugira melhorias.
