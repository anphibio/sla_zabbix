# SSO_LDAP.md

## LDAP/AD

Quando aplicável, suportar:

- Autenticação via LDAP
- Busca de usuário
- Busca de grupos
- Mapeamento de grupos para perfis

## SSO

Preparar para:

- SAML 2.0
- OIDC
- AD FS
- Keycloak
- Azure AD / Entra ID

## Regras

- Autorização deve continuar no sistema.
- Não confiar apenas em claims externas sem validação.
- Registrar login via SSO em auditoria.
