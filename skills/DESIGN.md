# DESIGN.md

## Objetivo visual

Criar interface corporativa moderna, limpa, responsiva e adequada para sistemas administrativos.

## Fonte de UX/UI

As regras detalhadas de UX/UI estão em:

- `IMPECCABLE_UI.md`
- `UI_COMPONENTS.md`

## Referências visuais

- Grafana
- GitLab
- Azure Portal
- Atlassian
- Linear
- OpenProject

## Estilo

- Profissional
- Corporativo
- Limpo
- Objetivo
- Pouco poluído
- Forte uso de hierarquia visual
- Boa leitura em telas grandes

## Layout padrão

- Sidebar lateral recolhível
- Topbar com usuário, notificações e tema
- Área principal com breadcrumbs
- Cards de resumo
- Tabelas com filtros
- Modais para ações rápidas
- Páginas de detalhe bem estruturadas

## Temas

O sistema deve suportar:

- Light mode
- Dark mode

## Regra

Toda alteração visual deve passar por `prompts/IMPECCABLE_REVIEW.md`.
